# 🍣 SushiZen - Landing Page

**SushiZen** é uma landing page elegante e moderna para um restaurante de sushi.  
A página oferece uma experiência agradável ao usuário, permitindo visualizar o cardápio, efetuar reservas e fazer pedidos para entrega.

## 🌐 Funcionalidades

- 📜 **Cardápio Interativo**  
  Visualize pratos organizados por categorias (sushis, sashimis, combinados, bebidas e sobremesas).

- 📅 **Reserva de Mesa Online**  
  Interface simples para selecionar data, horário e número de pessoas.

- 🛵 **Pedido Delivery**  
  Escolha itens do menu e faça seu pedido para entrega diretamente pelo site.

- 📱 **Design Responsivo**  
  Totalmente adaptado para dispositivos móveis, tablets e desktops.

- 🎨 **Estilo Personalizado**  
  Cores suaves, elementos orientais e imagens de alta qualidade para transmitir sofisticação e autenticidade japonesa.

## 🚀 Tecnologias Utilizadas

- HTML 5
- CSS3 
- JavaScript (Opcional/futura)
